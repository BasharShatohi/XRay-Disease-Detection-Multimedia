# Medical Image Processing - X-Ray Disease Localization

**Multimedia project focused on processing X-Ray images to detect and highlight areas affected by diseases.**

**An application for analyzing medical X-Ray images, applying filters, segmentation, and highlighting regions of interest (disease-affected areas).**

## ✨ Features

- Load and display X-Ray images
- Apply various image processing filters (contrast enhancement, noise reduction)
- Segmentation techniques to isolate affected regions
- Highlight disease areas with bounding boxes or masks
- Comparison between original and processed images
- Basic report generation
- User-friendly GUI with multiple tools

## 🛠 Technologies Used

- **Language**: C# / Python
- **Libraries**: OpenCV, Emgu CV, or similar
- **GUI**: Windows Forms / WPF
- **Image Processing**: Filtering, Edge Detection, Thresholding, Morphological Operations

## 🚀 How to Run

1. Open the project in Visual Studio
2. Build and run `MainForm`
3. Load an X-Ray image and start processing
